# ✅阿里的数据库能抗秒杀的原理

# 典型回答


其实，在阿里电商的秒杀等（据我了解，淘宝、天猫、猫超、大麦等都是这么干的）场景中，主要还是基于MySQL数据库在做扣减的，主要是因为这样做最可靠了（避免了redis扣减方案中的数据不一致、少卖等问题）。



但是我们都知道，数据库是抗不了热点行的并发更新的，于是阿里内部就对MySQL做了patch。



这个方案其实云上数据库RDS也支持了，所以我就可以讲了。（没开放的技术确实不敢讲，怕被请喝茶。。。）
